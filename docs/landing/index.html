Quick Start

Up and running in 60 seconds

No API keys, no external services. Just install and start coding.

Install

pip install opencontext-cli

Setup your project Default: context-first

cd your-project
opencontext setup --preset context-first

Context pack

opencontext pack . --query \
  "How does auth work?" --copy

Full agent setup

opencontext setup --preset full \
  --agent opencode
