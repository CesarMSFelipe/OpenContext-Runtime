# Roadmap

## Current Status
OpenContext Runtime has reached feature parity for its MVP release, including local indexing, repo maps, dependency graphs, retrieval, context packing, redaction, safe prompt assembly, local traces, memory usability primitives, and security scaffolds. All core subsystems are implemented and validated.

## Next Milestones
1. Expand parser-backed symbol extraction and deeper dependency graphs.
2. Optional public-key signing and transparency logs for workflow-pack integrity.
3. Production provider adapters (outside core package).
4. Hosted governance and multi-user policy enforcement scaffolds.

## Not Yet Enterprise Ready
The design is enterprise-oriented, but certification, multi-user policy enforcement, hosted governance, and provider-specific production adapters are not complete.
