# EU AI Act Notes

OpenContext Runtime is infrastructure for context governance and controlled
agentic execution. It does not itself provide a high-risk AI application.

## MVP Evidence

- Project inventory through manifests and repo maps.
- Traceability through OTel-compatible spans and local JSON traces.
- Security controls for prompt injection, secrets, providers, and tools.
- Evidence pack scaffold for governance review.

## Remaining Work

- Use-case classification guidance for downstream deployers.
- Human oversight workflow beyond approval-policy scaffolds.
- Formal post-market monitoring and incident workflow.
