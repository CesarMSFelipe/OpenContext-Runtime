# Contributing

The root [CONTRIBUTING.md](https://github.com/CesarMSFelipe/OpenContext-Runtime/blob/main/CONTRIBUTING.md) remains the canonical contributor guide: run `ruff check .`, `ruff format --check .`, `mypy packages/opencontext_core`, and `pytest` before PRs (or `make ci` for the full pipeline); keep core framework- and provider-agnostic; add tests for security-sensitive changes.
