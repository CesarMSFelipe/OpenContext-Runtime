# ISO 42001 Notes

OpenContext Runtime includes MVP controls that support AI management-system
evidence, but it is not ISO 42001 certified.

## Supporting Controls

- Documented system boundary and technology-agnostic core.
- Secure defaults for providers, tools, MCP, and traces.
- Traceable context selection and omission reasons.
- Local evidence artifacts for security scans and token efficiency.

## Gaps Before Audit Use

- Formal risk register ownership.
- Change-management workflow outside repository tests.
- Independent control testing.
- Retention policy enforcement beyond local trace sanitization.
