# NIST AI RMF Mapping Notes

This MVP maps OpenContext controls to NIST AI RMF functions for internal
governance planning. It is not a certification.

## Govern

- Fail-closed security modes.
- Provider policy registry.
- Tool and MCP disabled by default.
- Evidence pack scaffold.

## Map

- Data classification on context items and prompt sections.
- Project manifest and repo map inventory.
- Technology profile detection outside universal core.

## Measure

- Token reports and token-heavy file reporting.
- Trace spans for retrieval, ranking, packing, prompt assembly, and persistence.
- Security scan summaries without raw secret values.

## Manage

- ContextFirewall blocks unsafe exports, traces, and provider calls.
- Air-gapped mode blocks external providers and telemetry.
- Sanitized traces preserve decisions while removing raw content.
